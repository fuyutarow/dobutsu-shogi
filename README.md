# Dobutsu-shogi

## What's Dobutsu-shogi?

Dobutsu-shogi is a board game for two players. Each players' goal is to capture Lion.

## Requirement

- NodeJS 5.X+
- Typings 2.x+

## Install

### setup
```shell
npm install
typings install
```

### build
`npm run build`

### run
`python3 -m http.server 8000`
access localhost:8000/dist
